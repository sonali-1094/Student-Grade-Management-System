# Grade Management System

## Overview
A Java console application to manage student grades using arrays.
It allows adding students, calculating averages, assigning grades,
and generating performance reports.

## Features
- Menu-driven interface
- Input validation
- Grade categorization
- Performance analytics

## Technologies Used
- Java
- Arrays
- Control Flow
- Exception Handling

## How to Run
1. Open project in VS Code / IntelliJ
2. Compile all `.java` files
3. Run `GradeManagementSystem.java`
